# online code diff viewer

## setup
* npm package install
    ```
    npm install --registry=https://registry.npm.taobao.org
    ```
* pip package install
    ```
    pip install -r requirements.txt
    ```

## run
```
python run.py
```

## view
* `http://127.0.0.1:13579/`
* `http://127.0.0.1:13579/diff/`
